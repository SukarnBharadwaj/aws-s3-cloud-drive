
# S3 Bucker Drive App

Basic Web app to manage AWS s3 bucket over the web.

Allows the user to safely login and see the bucket content.

Also has an option to upload, download and view the present files.



## Deployment

To deploy this project run

```bash
  npm install
```

```bash
  npm run prod
```
